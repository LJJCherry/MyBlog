# MyBlog

### 简介

[hexo 官方网址](https://hexo.io/zh-cn/)
采用了[next](http://theme-next.iissnan.com/)主题

### 第一次使用

1、克隆项目到本地

2、npm i -g hexo-cli

### 日常开发

1、在 source/\_posts 中写入 markdown 文档

2、生成静态文件

    hexo generate

3 启动本地服务

    hexo server

4、部署到服务器

    hexo deploy

### 扩展链接

1、安装[next 主题](https://github.com/theme-next/hexo-theme-next)
